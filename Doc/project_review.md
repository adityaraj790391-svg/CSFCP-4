# Development Process

This document explains how the Number Guessing Game project was planned, developed, and managed using Visual Studio Code, Git, and GitHub.

---

## 1. Planning

- Selected the **Number Guessing Game** from the list of assignment options.
- Finalized **Python** as the programming language.
- Decided to use:
  - **Visual Studio Code** as the IDE
  - **Git** for version control
  - **GitHub** for remote repository hosting
- Identified basic features:
  - Generate a random number
  - Take user input
  - Compare guess with secret number
  - Show hints: "Too high" / "Too low"
  - Count the number of attempts

---

## 2. Environment Setup

1. Installed / verified **Python 3.x** on the system.
2. Installed **Visual Studio Code**.
3. Installed VS Code extensions (optional but helpful):
   - Python extension
   - GitLens (for Git integration)
4. Created a new project folder:

   ```bash
   mkdir number-guessing-game
   cd number-guessing-game
